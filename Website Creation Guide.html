GitHub par website banana shuru mein thoda mushkil lag sakta hai, lekin agar aap ye steps follow karoge to 10–15 minute mein apni pehli website live kar sakte ho.

Step 1: GitHub Account banao

  1. Browser mein https://github.com kholo.

  2. Sign up par click karo.

  3. Email, password aur username bhar kar account bana lo.

  4. Email verify kar lo.

Step 2: New Repository banao

  1. Login karne ke baad upar + icon par click karo.

  2. New repository select karo.

  3. Repository Name likho.

Example:

my-website

  1. Public select karo.

  2. Add a README file ko tick kar sakte ho.

  3. Create repository par click karo.

Step 3: HTML File Upload karo

  1. Repository ke andar Add file → Upload files par click karo.

  2. Apni file ka naam index.html hona chahiye.

  3. Agar file nahi bani hai, to Notepad kholo:

    • Aapka HTML code paste karo.

    • File → Save As

    • File name:

      index.html

    • Save as type:

      All Files

  4. Ab index.html GitHub par upload karo.

  5. Neeche Commit changes par click karo.

Step 4: GitHub Pages Enable karo

  1. Repository mein Settings tab kholo.

  2. Left side se Pages par click karo.

  3. Source mein:

    • Deploy from a branch select karo.

  4. Branch:

    • main

  5. Folder:

    • / (root)

  6. Save par click karo.

Step 5: Website Live hogi

1–2 minute wait karo.

Phir aapki website is address par khul jayegi:

https://YOUR_USERNAME.github.io/my-website/

Example:

https://vineetsingh.github.io/my-website/

Agar Website Update karni ho

  • index.html edit karo.

  • Nayi file upload karo ya GitHub ke editor se edit karo.

  • Commit changes karo.

  • 1–2 minute baad website automatically update ho jayegi.

Aapke liye sabse achha tareeka

Main aapko professional AI website banana sikha sakta hoon, jisme:

  • Home Page

  • AI Chat

  • AI Image Generator

  • Login / Signup

  • Dashboard

  • Payment System

  • Admin Panel

  • GitHub par upload

  • GitHub Pages par free hosting

Main ise Part 1 se Part 20 tak bilkul beginner-friendly tareeke se sikhaunga, jahan har step ke saath code aur explanation milegi.
